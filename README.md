# PublicLive2dPlug
一个简单的Live2D看板娘浏览器插件，让可爱的二次元角色陪伴你的浏览时光。

## 个人历时两周半开发的一款加载live2模型的浏览器插件！

## live2d看板娘助手
- [x] 加载插件后，任意网页加载live2d模型
- [x] 可以自定义增删改模型路径，模型名称（支持联系本人代理）
- [x] 实现自定义调节模型大小、高度、位置 

## 技术栈概述
- Live2D: 一种用于创建2D角色动画的技术。
- HTML/CSS/JavaScript: 用于前端界面设计和交互。
